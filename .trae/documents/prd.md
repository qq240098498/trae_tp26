## 1. 产品概述

衣物洗护管家 — 一款帮助用户科学管理衣物洗护信息的工具。用户可录入衣物信息（类型、品牌、材质、购买日期、价格），选择预置洗护方式，拍照留存洗护标签，洗前快速查看洗护要求，避免因错误洗涤导致衣物损坏。

- 目标用户：注重衣物保养的都市人群，拥有多材质衣物的家庭
- 核心价值：将散落的洗护标签信息数字化集中管理，降低洗护失误率，延长衣物寿命

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册，本地使用 | 录入、查看、编辑、删除衣物及洗护信息 |

### 2.2 功能模块
1. **衣物列表页**：展示所有已录入衣物，支持按类型/材质筛选，快速搜索，一键查看洗护要求
2. **衣物详情页**：展示衣物完整信息，包括洗护标签照片、洗护要求详情
3. **录入/编辑页**：录入衣物基本信息、选择洗护方式、拍照上传洗护标签

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 衣物列表页 | 顶部搜索栏 | 按衣物名称/品牌搜索 |
| 衣物列表页 | 分类筛选标签 | 按衣物类型（衣服/裤子/外套/毛衣等）和材质筛选 |
| 衣物列表页 | 衣物卡片列表 | 展示衣物缩略图、名称、品牌、洗护方式标签，点击进入详情 |
| 衣物列表页 | 新增按钮 | 点击进入录入页 |
| 衣物详情页 | 衣物基本信息 | 类型、品牌、材质、购买日期、价格 |
| 衣物详情页 | 洗护要求展示 | 以图标+文字方式展示选中的洗护方式 |
| 衣物详情页 | 洗护标签照片 | 展示拍摄的洗护标签图片，支持放大查看 |
| 衣物详情页 | 操作按钮 | 编辑、删除 |
| 录入/编辑页 | 基本信息 | 选择衣物类型、输入品牌、选择材质、购买日期、价格 |
| 录入/编辑页 | 洗护方式选择 | 从预置选项中多选：水洗/干洗、水温≤30°、不可漂白、不可烘干、熨烫温度 |
| 录入/编辑页 | 拍照上传 | 调用摄像头拍摄洗护标签照片，或从相册选择 |
| 录入/编辑页 | 保存/取消 | 保存录入信息或取消返回 |

## 3. 核心流程

用户打开应用 → 浏览衣物列表 → 点击"+"新增衣物 → 填写基本信息 → 选择洗护方式 → 拍摄洗护标签 → 保存 → 返回列表。洗衣服时 → 打开应用 → 搜索/筛选找到衣物 → 查看洗护要求 → 按要求洗涤。

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：暖白色（#FAF7F2）搭配深炭灰（#2D2D2D），点缀柔和的薰衣草紫（#8B7EC8）作为强调色，传递精致护理的质感
- 辅助色：浅卡其（#E8DFD0）、淡粉灰（#D4C5B9）
- 按钮风格：圆角胶囊形按钮，主按钮填充色，次按钮描边
- 字体：标题使用衬线字体（Playfair Display），正文使用无衬线字体（DM Sans），中文使用 Noto Serif SC / Noto Sans SC
- 布局风格：卡片式布局，顶部导航，柔和阴影营造层次感
- 图标风格：线性图标，配合洗护方式的符号化表达（水滴、温度计、禁止符号等）

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 衣物列表页 | 搜索栏 | 圆角输入框，搜索图标，浅色背景 |
| 衣物列表页 | 筛选标签 | 横向滚动的胶囊标签，选中态填充色 |
| 衣物列表页 | 衣物卡片 | 圆角卡片，左侧缩略图，右侧信息+洗护图标标签 |
| 衣物列表页 | 新增按钮 | 右下角浮动圆形按钮，"+"图标 |
| 衣物详情页 | 信息区域 | 分组展示，标签式呈现材质和洗护方式 |
| 衣物详情页 | 标签照片 | 横向滚动的照片卡片，点击放大 |
| 录入/编辑页 | 表单 | 分段式表单，下拉选择+输入框，图标化洗护方式选择 |
| 录入/编辑页 | 拍照区域 | 虚线框拍照区域，支持点击拍照或选择图片 |

### 4.3 响应式设计
- 桌面优先设计，最大宽度 1200px 居中
- 平板：双列卡片布局
- 手机：单列卡片布局，底部浮动新增按钮
- 触屏优化：卡片点击区域≥44px，按钮间距充足

### 4.4 数据持久化
- 使用 localStorage 存储衣物数据
- 照片使用 Base64 编码存储于 localStorage（单张限制 500KB 以内，自动压缩）
- 导出/导入功能支持 JSON 格式备份
